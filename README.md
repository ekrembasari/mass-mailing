# Mass Mailing
Send mass email via python with **HTML** body text support.

  * **"ms.py"** file stores HTML body part of mail. 
  * **"email.xlsx"** stores list of emails. 

  * **"script.py"** is main script. 

  * **"config.py"** is configuration file
